# Rekha Nahata Portfolio

Modernized static portfolio site built with pure HTML, CSS, and minimal JavaScript.

## Project Structure

- `index.html` - home page with Hero, About, Work, and Contact sections.
- `projects/` - individual project pages.
- `css/styles.css` - design system, responsive layout, animations, project gallery/lightbox styles.
- `js/script.js` - theme toggle, reveal animation, contact form UX, project-page gallery/lightbox logic.
- `assets/images/` - project thumbnails and gallery images.
